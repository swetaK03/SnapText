# 🚀 SnapText

SnapText is a modern AI-powered web application built using Next.js and Firebase that allows users to extract and process text efficiently. It provides a clean UI, real-time processing, and seamless integration with cloud services.

---

## 🌟 Features

* 🔍 OCR-based text extraction
* 🤖 AI-powered text processing
* 🔐 Firebase authentication integration
* ⚡ Real-time data handling
* 🎨 Modern responsive UI (Tailwind CSS)
* ☁️ Cloud-based backend support

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Firebase (Firestore, Auth)
* **AI Integration:** Genkit / AI flows
* **Deployment:** Vercel

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/swetak03/SnapText.git
cd SnapText
```

### 2️⃣ Install dependencies

```
npm install
```

## ▶️ Run the project

```
npm run dev
```

Open:

```
http://localhost:3000
```
